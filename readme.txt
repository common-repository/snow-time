== Description ==

Animated snow plugin for your website.

Documentation
To change global settings, go to "Settings" -> "Snow Time".

== Installation ==

Installing "Snow Time" can be done either by searching for "Snow Time" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
